- This project has been developped under Visual Studio Code 2019
- Compile project using make:  

  ```bash
  cd ./AP4A_Submarine_IOT/Debug; make all
  ```

- Run the project:  

  ```bash
  ./AP4A_Submarine_IOT/Debug/AP4A_Submarine_IOT.exe
  ```
